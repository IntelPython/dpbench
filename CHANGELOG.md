## [0.90.0] - 2023-06-23

### Added
* First release.
* Infrastructure to run benchmarks in independent threads.
* Frameworks
    * numpy
    * python
    * dpnp
    * dpcpp
    * numba
    * numb-dpex
* Benchmarks
    * black_scholes
    * dbscan
    * gpairs
    * kmeans
    * knn
    * l2_norm
    * pairwise_distance
    * pca
    * rambo
* Experimental framework
    * numba-mlir
* Experimental benchmarks
    * multiple npbench benchmarks
    * multiple polybench benchmarks
